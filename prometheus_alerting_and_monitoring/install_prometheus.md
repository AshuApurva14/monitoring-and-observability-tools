## Installaton of Prometheus

- Below are the simple and easy steps to install and setup `Prometheus` server with their components:

```bash
sudo apt update
sudo apt install prometheus

```

- Once installation is completed then, check whether the process is running or not

```bash
sudo systemctl status prometheus

```
